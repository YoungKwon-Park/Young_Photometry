# Young's Fiber Photometry Project
the only archive used for Fiber Photometry project
# Project Summary
Built an end-to-end spectrometer-based photometry workflow that integrates acquisition, raw-data reconstruction, MATLAB-to-Python interoperability, event-aligned signal analysis, and real-time spectral correction for dual-color measurements.

# What problem was solved
- heterogeneous devices
- fragmented raw outputs
- dual-color signal overlap
- need for real-time correction
- reusable acquisition-to-analysis workflow

# Contribution
- Designed the acquisition-layer architecture in LabVIEW
- Structured raw logging for downstream reconstruction
- Standardized fragmented outputs into reusable session datasets
- Built MATLAB-to-Python transfer workflow
- Implemented event-aligned normalization and quantification
- Designed and validated a real-time spectral correction algorithm
- Extended the validated pipeline to dopamine/serotonin sensor data
